# To Do List

## 프로젝트 소개
JavaScript로 만든 인터랙티브 할 일 목록(To Do List) 웹 앱입니다.
할 일을 추가하고, 완료 처리하며, 수정·삭제할 수 있습니다.
localStorage를 사용하여 새로고침 후에도 데이터가 유지됩니다.

## 주요 기능
- 할 일 추가 (추가 버튼 / Enter 키)
- 할 일 목록 화면 표시 (innerHTML)
- localStorage를 이용한 데이터 영구 저장 (JSON.stringify, JSON.parse)
- 완료 토글 (classList, CSS 스타일 변경)
- 할 일 삭제
- 할 일 수정 (prompt)

## 사용 기술
- HTML5
- CSS3
- JavaScript (DOM 조작, localStorage)
- Git / GitHub
## 실행 방법
1. 저장소를 clone 하거나 프로젝트 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. 할 일을 입력한 뒤 추가, 완료, 수정, 삭제 기능을 사용합니다.

## 파일 구조
| 파일 | 설명 |
|------|------|
| index.html | 페이지 구조 (입력창, 버튼, 목록 영역) |
| style.css | 스타일 (레이아웃, 완료 시 취소선 등) |
| script.js | 기능 구현 (추가, 표시, 저장, 완료, 삭제, 수정) |
